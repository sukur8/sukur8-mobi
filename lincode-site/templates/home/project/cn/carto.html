Carto

主题

Carto是一个关于地理信息的网络应用。它动态显示了一个法国城市，图卢兹市的交通状况。开发过程中使用了 XML/XSLT, SVG, Javascript 和 AJAX 等技术。

网页

你可以访问 Carto。
你需要使用 Firefox，Opera 或者 IE6，以获得游览器对于 SVG 的支持。如果你只有 IE 5，请下载一个为获得 SVG 支持的浏览器插件：Adobe Inc.。

注意演示中 ajax 部分功能没有移植到 Google app engine server。但在 PHP 服务器下运行正常。
