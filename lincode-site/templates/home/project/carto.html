Carto

Subject

Carto is a cartographic web application for dispaly dynamically the traffic information of the french city Toulouse. I used XML/XSLT, SVG, Javascript and AJAX during this project.

Web Site

You can visit the Carto.
You need Firefox, Opera or IE 6 for support SVG on browser. If you have only IE 5, please download a plugin from Adobe Inc. .

Attention: the function of ajax is not realized on Google app engine in the demo. You can find it in the version for PHP server.
